# Jurassic Left Behind — Dossier Reader

A literary webnovel reader for *Jurassic Left Behind* — a 1993 film-continuity Jurassic Park side-story.

**Live:** https://dknos.github.io/jurassic-left-behind

## About
Mateo Valverde was a temporary sanitation contractor on Isla Nublar. His badge was valid for fourteen days. When the park's computer system performed a full reboot, it reverted to a saved state. Temporary badges were not in the saved state.

Thirty chapters. After Michael Crichton.

## Reader features
- **Dossier aesthetic** — declassified-file UI with stamps, redactions, and evidence sidecars  
- **Mobile + desktop** — auto-switches at 900px, or override via the AUTO/MOBILE/DESKTOP pill
- **Light / Archive / Blackout** themes  
- **Accent colors** — Alarm Red, Caution Amber, Diagnose Cyan, Signal Chartreuse  
- **Glossary** — tap highlighted terms to open a definition drawer  
- **Redaction reveal** — tap black blocks in prose to reveal redacted InGen terminology  
- **Evidence sidecar** — WHO / WHERE / WHAT / WHEN cards per chapter  
- **Tweaks panel** — font size, line height, face, justify, stamp intensity, paper texture

## Stack
Static HTML + React 18 (CDN) + Babel Standalone. No build step. Works on GitHub Pages.
